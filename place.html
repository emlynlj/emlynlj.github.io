Step 1 - Install TamperMonkey (chromium browsers)
Step 1.5 - If you are using Firefox use violentmonkey

Step 2 - Install the script to Tamper/ViolentMonkey (click the install button!)

Step 3 - Make sure it is enabled. Now, load up r/place and you should see a little menu icon pop up in the top left hand corner of your screen. If you don't, contact me (@EmlynLJ on discord)

Step 4 - Click the menu icon. In the section that says Template URL, first enter https://github.com/SpaceRPlace/Overlay/raw/main/Template and click "Always Load." If a window pops up, click the button that says something like "Allow domain".

Step 5 - Reload the template (button at the top). you are now ready to go!
